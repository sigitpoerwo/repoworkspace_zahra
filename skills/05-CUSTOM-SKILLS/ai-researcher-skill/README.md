# AI-Researcher Skill

Autonomous AI system for automated scientific research - from literature review to paper generation.

## Quick Start

```bash
# Invoke skill
/ai-researcher

# Or use Skill tool
Skill(skill: "ai-researcher")
```

## What It Does

- 📚 Automated literature review and synthesis
- 💡 Research idea generation from references
- 🔬 Algorithm design and implementation
- 🧪 Automated experimentation and validation
- 📝 Full academic paper generation
- 📊 Result analysis and visualization

## For Lecturers

### Teaching Use Cases
- Demonstrate research methodology workflows
- Generate baseline code for student projects
- Create paper templates for writing workshops
- Show literature review techniques

### Research Use Cases
- Accelerate paper drafting for conferences
- Generate proof-of-concept for grant proposals
- Explore multiple research directions in parallel
- Assist student thesis supervision

## Setup

```bash
# Clone repository
git clone https://github.com/HKUDS/AI-Researcher.git
cd AI-Researcher

# Setup with Docker
docker-compose up -d

# Or manual setup
pip install -r requirements.txt

# Configure API keys
cp .env.example .env
# Add your OpenRouter or GitHub AI API key

# Launch
python app.py
```

## Important Notes

⚠️ **Always review AI output before using**
- Validate algorithm correctness
- Verify experimental results
- Check paper logic and citations

✅ **Use as assistant, not replacement**
- Critical thinking remains essential
- Domain expertise still required
- Maintain academic integrity

## 🚀 How to Use

### Finding Skills

```bash
# Browse by category
cd JONI-SKILLS-LIBRARY/01-SIAP-PAKAI/coding

# View inventory
cat JONI-SKILLS-LIBRARY/INVENTARIS.md

# Check category README
cat JONI-SKILLS-LIBRARY/01-SIAP-PAKAI/README.md
```

### Using Skills

```bash
# Activate any skill
/skill-name

# Examples:
/content-marketer    # Writing
/expert-frontend     # Coding
/ddg-research        # Research
/core-brainstorm     # AI Agent
```

### Adding New Skills

```bash
# 1. Put in BELUM-DICOBA first
mv new-skill JONI-SKILLS-LIBRARY/03-BELUM-DICOBA/dari-clawhub/

# 2. Test it
/new-skill

# 3. Move to appropriate category
mv JONI-SKILLS-LIBRARY/03-BELUM-DICOBA/dari-clawhub/new-skill \
   JONI-SKILLS-LIBRARY/01-SIAP-PAKAI/category/

# 4. Update INVENTARIS.md
```
